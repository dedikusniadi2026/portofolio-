### Langkah-langkah

1.  **Clone Repositori:**
    ```bash
    git clone https://github.com/dedikusniadi2016/portofolio-.git
    cd portofolio-
    ```

2.  **Instal Dependensi:**
    ```bash
    npm install  # atau yarn install
    ```
3.  **Jalankan Aplikasi:**
    ```bash
    npm run dev # atau npm start
    ```
    Aplikasi akan berjalan di `http://localhost:3000`
